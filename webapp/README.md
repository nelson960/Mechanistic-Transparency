# Web App Starter

This folder is a standalone frontend that loads exported JSON bundle files.

## Expected input

Recommended: `viewer_payload.json` generated by:

```bash
python scripts/build_interactive_model_viewer.py \
  --prompt "The secret code is 73914. Repeat the secret code exactly:" \
  --prompt-id my_prompt_001 \
  --task copy \
  --model gpt2-small \
  --out outputs/viewer_payload.json
```

## Run locally

Serve with any static server and open `index.html`.

Example:

```bash
cd webapp
python3 -m http.server 8000
```

Then open `http://localhost:8000` and upload your `viewer_payload.json`.

## 3D viewer controls

- Left drag: rotate
- Right drag (or Shift + left drag): pan
- Mouse wheel: zoom
- Click a node: inspect it
- Click a component/path row: jump selection in the 3D graph
